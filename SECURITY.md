# Security Policy

## Supported Versions

The following versions of `testing-maps` are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within `testing-maps`, please send an e-mail to the maintainer.

Please include:
- A description of the vulnerability.
- Steps to reproduce it.
- Possible impacts.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
